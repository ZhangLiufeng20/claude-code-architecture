📋 项目概览 01

Claude Code 是 Anthropic 开发的一款终端 AI 编程助手，它运行在命令行中，能够直接读写文件、执行 Shell 命令、搜索代码、与 Git 交互，本质上是一个具备完整系统操作权限的 AI Agent。

技术栈分析

技术用途说明
TypeScript + TSX核心语言全部源码均为 TS/TSX，强类型设计
Bun运行时 & 打包器使用 bun:bundle 特性门控实现 DCE
React (Ink)终端 UI 渲染用 React 组件树渲染终端界面
Zod v4Schema 验证所有工具输入/输出通过 Zod 校验
Anthropic SDKAPI 通信与 Claude API 交互的官方 SDK
MCP Protocol工具扩展Model Context Protocol 实现外部工具集成
Lodash-es工具函数memoize, uniqBy 等常用函数
🔑 关键发现：Claude Code 本质是一个用 React 渲染终端 UI 的 Agent Loop —— 不断调用 AI API、执行工具、收集结果、再调用 API，直到任务完成或用户中断。

🏗️ 整体架构分层 03

Claude Code 的架构可以清晰地划分为 6 个核心层次，从用户输入到 AI 响应形成了一条完整的数据流水线：

核心数据流

1 用户在终端输入 prompt 或执行 /command
↓
2 processUserInput() 解析输入，判断是命令还是查询
↓
3 QueryEngine.submitMessage() 构建 System Prompt + 上下文
↓
4 query() 进入异步生成器循环：调用 API → 流式接收 → 执行工具
↓
5 工具执行经过权限检查 → Hook 系统 → 实际执行 → 返回结果
↓
6 循环直到模型返回 end_turn / 超出预算 / 用户中断
